# zed-less

LESS language support for [Zed](https://zed.dev).

## Features

- Syntax Highlighting
- Completion(uses vscode-css-language-server for completion)

## Grammar

- [tree-sitter-less](https://github.com/jimliang/tree-sitter-less)
